# Security Policy

## Reporting

Please do not open public issues for sensitive vulnerability details, credentials, personal data, or private artifacts.

Preferred channels:

- Email: naees@protonmail.com
- PGP public key: https://naees.github.io/NaeesSecurity/keys/publickey.naees.asc
- Signal: @naeessec

For non-sensitive website issues such as typos, broken links, or rendering bugs, GitHub Issues are acceptable.

## Handling

- Encrypt sensitive vulnerability details when possible.
- Include only the minimum context needed to begin triage.
- Redact personal, confidential, regulated, or third-party data unless explicitly authorized to share it.
- Do not perform testing against systems unless you have permission.
